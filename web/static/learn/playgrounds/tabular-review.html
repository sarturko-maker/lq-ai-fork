Failed-cell demo

Tabular Review runs the same per-position extraction over many documents at once. Retrieval is lexical full-text search (FTS) over each document's chunks, top-K = 4; one structured-output LLM call per cell returns {value, cited_chunk_indices, confidence, justification}. Click any cell to open the per-cell citation drawer.
